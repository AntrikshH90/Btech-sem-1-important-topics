**The Reality:**
*   PPS is 50% "Write a Program" and 50% "Theory".
*   **Big 2 Topics** cover the mandatory programs.
*   **Section A** asks for tricky syntax/output questions.

---

## üõ†Ô∏è Unit 1: Basics
### üü¢ The "Big 2" (For Long Questions - 20 Marks)
1.  **Block Diagram:** Draw CPU, ALU, CU, Memory, Input, Output clearly.
2.  **Operators:** Explain Arithmetic, Relational, Logical, Bitwise with examples.

### ‚ö†Ô∏è The "Section A" Risk (Memorize These)
*   **Compiler vs Interpreter:** Compiler = Whole Code (Fast). Interpreter = Line by Line (Slow).
*   **Ternary Operator:** Syntax `(condition) ? true_val : false_val;`.
*   **Storage Classes:** `auto`, `extern`, `static`, `register`. (Just names).

---

## üõ†Ô∏è Unit 2: Control Flow
### üü¢ The "Big 2" (For Long Questions - 20 Marks)
1.  **Switch Case:** Program for Calculator (+, -, *, /) using switch.
2.  **Loops:** Pattern Printing (Pyramid OR Diamond).

### ‚ö†Ô∏è The "Section A" Risk (Memorize These)
*   **Break vs Continue:** `break` exits loop; `continue` skips to next iteration.
*   **Implicit vs Explicit:** Auto conversion (`int` to `float`) vs Force cast `(int)`.
*   **Else-If Ladder:** Syntax flow.

---

## üõ†Ô∏è Unit 3: Arrays
### üü¢ The "Big 2" (For Long Questions - 20 Marks)
1.  **Matrix Multiplication:** The logic `c[i][j] += a[i][k] * b[k][j]`.
2.  **Sorting:** Code for Bubble Sort OR Insertion Sort (Unit 4/5 syllabus mix).

### ‚ö†Ô∏è The "Section A" Risk (Memorize These)
*   **String Termination:** Strings end with `\0` (Null Character).
*   **2D Array Decl:** `int arr[rows][cols];`.
*   **Linear Search:** Simple `for` loop to find `key`.

---

## üõ†Ô∏è Unit 4: Functions
### üü¢ The "Big 2" (For Long Questions - 20 Marks)
1.  **Recursion:** Program for Factorial `fact(n) = n * fact(n-1)` OR Fibonacci.
2.  **Parameter Passing:** Explain Call by Value vs Call by Reference with Swap example.

### ‚ö†Ô∏è The "Section A" Risk (Memorize These)
*   **Prototype:** Declaration before main `void fun(int);`.
*   **Formal vs Actual:** Formal = In function def. Actual = Passed during call.
*   **Return Type:** Use `void` if nothing returned.

---

## üõ†Ô∏è Unit 5: Advanced (Structure/File)
### üü¢ The "Big 2" (For Long Questions - 20 Marks)
1.  **File Handling:** Program to copy contents of `file1` to `file2`.
2.  **Structure:** Difference between Structure vs Union (Diagram needed).

### ‚ö†Ô∏è The "Section A" Risk (Memorize These)
*   **Pointer Operators:** `&` (Address of), `*` (Value at).
*   **File Modes:** "r" (read), "w" (overwrite), "a" (append).
*   **Macros:** `#define PI 3.14` (Text replacement).
