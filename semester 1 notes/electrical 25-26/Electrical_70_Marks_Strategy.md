## üõë Phase 1: The "No-Miss" Topic Checklist

### **Unit 1: DC Circuits (The Foundation)**
- [ ] **Ohm's Law:** Statement, Limitations.
- [ ] **Active/Passive Elements:** Definition & Examples (Battery vs Resistor).
- [ ] **Linear/Non-Linear Elements:** Definition & Examples.
- [ ] **Unilateral/Bilateral Elements:** Definition & Examples (Diode vs Resistor).
- [ ] **Kirchhoff‚Äôs Laws:** KCL (Charge conservation) & KVL (Energy conservation).
- [ ] **Mesh Analysis:** Concept, Super-mesh (current source shared by two meshes).
- [ ] **Nodal Analysis:** Concept, Super-node (voltage source between two non-reference nodes).
- [ ] **Star-Delta Transformation:** Formulas for both conversions (Derivation not needed, but formula MUST be memorized).
- [ ] **Superposition Theorem:** Statement, Steps, Limitation (Linear networks only).
- [ ] **Thevenin‚Äôs Theorem:** Statement, $V_{th}$, $R_{th}$ calculation.
- [ ] **Norton‚Äôs Theorem:** Statement, $I_{sc}$, $R_{N}$ calculation.
- [ ] **Source Transformation:** Voltage source $\leftrightarrow$ Current source.

### **Unit 2: AC Circuits (The Conceptual Core)**
- [ ] **AC Fundamentals:** RMS Value, Average Value, Peak Factor, Form Factor (Values for Sine wave).
- [ ] **Phasor Algebra:** j-operator, Rectangular to Polar conversion.
- [ ] **Pure Circuits:** Pure R (In-phase), Pure L (Lag by 90¬∞), Pure C (Lead by 90¬∞).
- [ ] **RL Series Circuit:** Impedance triangle, Phasor diagram, Power Factor.
- [ ] **RC Series Circuit:** Impedance triangle, Phasor diagram.
- [ ] **RLC Series Circuit:** Impedance formula, Case of $X_L > X_C$ vs $X_C > X_L$.
- [ ] **Series Resonance:** Derivation of $f_r$, Bandwidth, Quality Factor, Resonance Curve.
- [ ] **Parallel Resonance:** Condition for resonance (basic idea).
- [ ] **3-Phase Systems:** Difference between 1-$\phi$ and 3-$\phi$.
- [ ] **Star Connection:** Relation between Line/Phase Voltage & Current ($V_L = \sqrt{3}V_{ph}$).
- [ ] **Delta Connection:** Relation between Line/Phase Voltage & Current ($I_L = \sqrt{3}I_{ph}$).
- [ ] **Power in 3-Phase:** $P = \sqrt{3}V_L I_L \cos\phi$ (Active), $Q = \sqrt{3}V_L I_L \sin\phi$ (Reactive).

### **Unit 3: Transformers (The Scoring Unit)**
- [ ] **Construction:** Core type vs Shell type, Berry type (basic).
- [ ] **Working Principle:** Faraday's Laws, Mutual Induction.
- [ ] **Ideal Transformer:** Assumptions, Phasor diagram (No load).
- [ ] **Practical Transformer:** Equivalent Circuit, Phasor diagram (Lagging/Leading PF).
- [ ] **EMF Equation:** **Derivation is mandatory**.
- [ ] **Voltage Regulation:** Definition, Formula.
- [ ] **Efficiency:** Formula, Condition for Max Efficiency (Derivation).
- [ ] **Losses:** Hysteresis (Formula), Eddy Current (Formula).
- [ ] **Tests:** Open Circuit (Finds Iron Loss), Short Circuit (Finds Copper Loss).

### **Unit 4: Machines (The Mechanics)**
- [ ] **DC Machine Construction:** Yoke, Poles, Armature, Commutator, Brushes.
- [ ] **DC Generator:** EMF Equation (Derivation), Types (Series, Shunt, Compound).
- [ ] **DC Motor:** Working Principle (Lorentz Force), Back EMF, Torque Equation ($T \propto \phi I_a$).
- [ ] **3-Phase Induction Motor (IM):** Construction (Squirrel Cage vs Slip Ring).
- [ ] **RMF:** Concept of Rotating Magnetic Field (Speed $N_s$).
- [ ] **Slip:** Definition, Formula, Numerical.
- [ ] **Torque-Slip Characteristics:** **Graph is crucial**, Starting torque, Max torque.
- [ ] **Applications:** Uses of Shunt/Series DC motors, Induction motors.
- [ ] **Synchronous Motor:** Basic principle, Synchronous speed.

### **Unit 5: Installations (The Theory Buffer)**
- [ ] **Switch Fuse Unit (SFU):** Diagram & Purpose.
- [ ] **MCB vs MCCB:** Difference & Applications.
- [ ] **ELCB:** Working principle (Safety against shock).
- [ ] **Earthing:** Purpose, Pipe Earthing vs Plate Earthing (Detailed diagrams).
- [ ] **Batteries:** Primary vs Secondary, Lead-Acid Battery (Chemical reaction not deep, but construction).
- [ ] **Wires & Cables:** Types (basic).
- [ ] **Energy Consumption:** Calculation (Units = kWh).

## üéØ Phase 2: The "70 Marks" Exam Strategy

### **Section A: The "Zero Error" Zone (20 Marks)**
*   **Goal:** 10/10 Correct.
*   **Risk:** Simple definition mistakes or forgetting units.
*   **Strategy:**
    *   Answer in **2-3 bullet points**. Never write paragraphs.
    *   **Always** write the formula if applicable (e.g., Define Q-factor -> Write definition + Formula).
    *   **Always** draw a small circuit/diagram if possible (e.g., KCL -> Draw a node with arrows).

### **Section B: Accurately Selected Derivations (30 Marks)**
*   **Goal:** Attempt 3 questions. Choose **Derivations** over Numericals if you are unsure about calculation.
*   **Best Choices:**
    1.  **Resonance Derivation** (Unit 2) - Standard steps, hard to mess up.
    2.  **Transformer EMF / Efficiency** (Unit 3) - Very logical.
    3.  **Torque-Slip Characteristics** (Unit 4) - Mostly graphical/theory.
    4.  **Star/Delta Derivation** (Unit 2) - Phasor diagram based.
*   **Avoid:** Complex DC circuit numericals in this section *unless* you are 100% confident (sign errors are fatal here).

### **Section C: The "Perfect Presentation" Zone (20 Marks)**
*   **Goal:** Attempt 1 from each unit.
*   **Protocol for 7 Marks:**
    1.  **Heading:** Start with a clear heading.
    2.  **Principal/Statement:** 2 lines.
    3.  **Diagram:** Draw a large, neat diagram (use pencil/scale). **Labels are mandatory.**
    4.  **Derivation/Explanation:** Step-by-step with equation numbers.
    5.  **Conclusion/Result:** Box the final formula.
    6.  **Numerical (if part of Q):**
        *   Given: ...
        *   Formula: ...
        *   Calculation: ...
        *   Answer: **[Boxed with Unit]**

## ‚ö†Ô∏è Phase 3: The Danger Zones (Where Students Lose Marks)
1.  **Calculator Mode:** Ensure it's in Degrees, not Radians (for AC circuits `sin(30)`).
2.  **Units:** Forgetting 'Hz', 'V', 'A', 'W', 'VAR' costs 0.5 - 1 mark.
3.  **Phasor Diagrams:** Drawing arrows in wrong direction (Lagging vs Leading).
4.  **Induction Motor:** Confusing $N$ (Rotor speed) and $N_s$ (Sync speed).
5.  **Efficiency:** Forgetting to convert kW to W or not multiplying by Power Factor ($kW = kVA \times pf$).
